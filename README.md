# Sales Prediction using python 

## Project Overview

This project predicts product sales based on advertisement spending on different platforms such as TV, Radio, and Newspaper using Machine Learning techniques in Python.

The model helps businesses estimate future sales by analyzing advertising budgets.

---

## Dataset

The dataset used in this project is:

* Advertising.csv

Dataset contains:

* TV advertising budget
* Radio advertising budget
* Newspaper advertising budget
* Sales data

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

---

## Machine Learning Algorithm

* Linear Regression
* Random Forest Regressor (Optional comparison)

---

## Project Workflow

1. Import Libraries
2. Load Dataset
3. Data Cleaning
4. Data Visualization
5. Feature Selection
6. Train-Test Split
7. Model Training
8. Sales Prediction
9. Model Evaluation
10. Data Visualization of Predictions

---

## Data Visualization

The project includes:

* Correlation Heatmap
* Scatter Plots

---

## Model Accuracy

The model achieved high prediction accuracy using Linear Regression with a strong R² score.

---

## Conclusion

The project successfully predicts sales based on advertising expenditure. TV advertising has the highest impact on sales compared to Radio and Newspaper advertisements.
